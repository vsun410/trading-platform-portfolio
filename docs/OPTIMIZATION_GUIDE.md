# 자본 배분 최적화 가이드

## 1. Kelly Criterion

기대 자본 성장률 최대화

```
f* = (p × b - q) / b
```
- p: 승률
- q: 패배율 (1-p)
- b: 승리/손실 비율

**실무 적용:**
- Half Kelly: 50% 사용 (변동성 감소)
- Quarter Kelly: 25% 사용 (보수적)

## 2. Mean-Variance (마코위츠)

목표 수익률에서 분산 최소화

**제약 조건:**
- 가중치 합 = 100%
- 최소 기대 수익률 충족
- 각 가중치 ≥ 0 (공매도 금지)

## 3. Risk Parity

각 자산의 위험 기여도 동일하게 배분

## 리밸런싱 트리거

| 방법 | 설명 |
|:---|:---|
| 임계값 | 목표 대비 5% 이상 이탈 시 |
| 시간 | 주간/월간 정기 |
| 이벤트 | 특정 이벤트 발생 시 |

## 주의사항

1. **과최적화**: In-Sample 최적화 → Out-of-Sample 성능 저하
2. **추정 오차**: 과거 수익률/공분산 추정의 불확실성
3. **거래 비용**: 잦은 리밸런싱 = 비용 증가
