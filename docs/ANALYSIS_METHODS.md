# 분석 방법론

## 성과 지표

### 1. Sharpe Ratio
```
Sharpe = (Rp - Rf) / σp
```
- Rp: 포트폴리오 수익률
- Rf: 무위험 수익률 (3%)
- σp: 표준편차

| 값 | 해석 |
|:---:|:---|
| < 0 | 나쁨 |
| 0~1 | 보통 |
| 1~2 | 좋음 |
| > 2 | 매우 좋음 |

### 2. Maximum Drawdown (MDD)
피크에서 저점까지 최대 하락폭

| 값 | 해석 |
|:---:|:---|
| < 10% | 낮은 리스크 |
| 10~20% | 보통 |
| > 20% | 리스크 |

### 3. Value at Risk (VaR)
95% 신뢰수준에서 최대 예상 손실

## 상관관계 분석

### 피어슨 상관계수
| 값 | 해석 |
|:---:|:---|
| 0.7~1.0 | 강한 양의 상관 |
| 0.3~0.7 | 중간 양의 상관 |
| -0.3~0.3 | 약한 상관 |
| -0.7~-0.3 | 중간 음의 상관 |
| -1.0~-0.7 | 강한 음의 상관 |

**포트폴리오 관점:**
- 낮은 상관관계 전략 조합 → 분산 효과
- 음의 상관관계 전략 → 헤지 효과
